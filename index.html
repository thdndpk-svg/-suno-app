AI Music Auto Studio v20

엔터테인먼트 제작 스타일 UI · 한국어 옵션 구성 · Suno용 Title / Style / Lyrics 자동 생성 · 빠른 붙여넣기 작업 흐름

실행 → 자동분석 → 노래 제작 → Suno 제작 → 노래 편집 시작

프로젝트 설정

상태: 대기중
사용 방법
1. 음성 버튼을 누르고 말하거나, 텍스트 창에 직접 입력합니다.
2. 실행 버튼을 누르면 AI가 명령을 분석해 장르, 분위기, 보컬, 가사 언어 등을 자동 반영합니다.
3. 필요하면 기본 설정을 손으로 수정한 뒤 노래 제작 버튼을 누릅니다.
4. 생성된 Title / Style / Lyrics를 Suno에 붙여넣습니다.
5. Suno에서 완성된 mp3를 지정 폴더에 넣고 노래 편집 시작을 누릅니다.

입력 예시
- 비 오는 날 듣기 좋은 발라드 여성보컬 10곡 만들어줘
- 카페 음악 느낌의 가사 없는 연주곡 12곡 만들어줘
- 중간에 랩이 들어가는 힙합 남성보컬 8곡 만들어줘
- 영어 노래로 밝고 경쾌한 팝 6곡 만들어줘

곡 수정 예시
- 1번곡 조금 더 경쾌하게 만들어줘
- 2번곡 더 슬프게 바꿔줘
- 3번곡 더 따뜻한 느낌으로 수정해줘
- 4번곡 더 강렬하게 바꿔줘

빠른 복사 기능
Next Copy는 순서대로
1번곡 Title → 1번곡 Style → 1번곡 Lyrics → 2번곡 Title → 2번곡 Style → 2번곡 Lyrics
순으로 복사합니다.

참고
- 장르, 분위기, 보컬 타입은 모두 한국어로 표시되지만, 내부 Style은 Suno 인식을 위해 영어 형식으로 생성됩니다.
- 현재는 UI와 생성 로직 중심 단계이며, 실제 mp3 편집 엔진은 별도 연결 단계가 필요합니다.
